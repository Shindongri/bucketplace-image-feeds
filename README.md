Getting Started

## Start

In the project directory, you can run:

### `yarn start`

## Build

Builds the app for production to the `build` folder.<br>

Your app is ready to be deployed!

### `npm run build`

## Following technologies are used

- typescript
- axios
- react
- react-dom
- styled-components
- react-spinners
